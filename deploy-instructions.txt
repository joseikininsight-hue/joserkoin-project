=================================================================
Tailwind CSS Build Edition - 簡単デプロイ手順
=================================================================

【ステップ1】PRをマージ
-----------------------------------------------------------------
1. ブラウザで以下のURLを開く：
   https://github.com/joseikininsight-hue/joserkoin-project/pull/1

2. 「Merge pull request」ボタンをクリック

3. 「Confirm merge」をクリック

【ステップ2】サーバーで更新を取得
-----------------------------------------------------------------
サーバーにSSHでログインして、以下のコマンドを実行：

cd /path/to/wordpress/wp-content/themes/grant-insight-perfect/
git pull origin main

※ /path/to/wordpress/ は実際のWordPressのパスに置き換えてください

【ステップ3】確認
-----------------------------------------------------------------
1. WordPressサイトをブラウザで開く

2. デザインが正常に表示されているか確認

3. ブラウザの開発者ツール（F12）を開いて、
   Networkタブで「tailwind-build.min.css」が
   読み込まれているか確認

=================================================================

【代替方法】FTPでアップロード
-----------------------------------------------------------------
Gitが使えない場合は、以下のファイルをFTPでアップロード：

必須ファイル:
✓ functions.php
✓ style.css
✓ assets/css/tailwind-build.min.css
✓ assets/css/tailwind-build.css

アップロード先:
/wp-content/themes/grant-insight-perfect/
